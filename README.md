# JavaScriptBootCampTask1
mLab CodeTribe JavaScript Assignments

## Packages Used

This project uses the following npm package:

- **readline-sync** (^1.4.10) - Used in the conditional statements and loops exercises for synchronous readline functionality

## Installation

To install dependencies:

```bash
cd conditional_statements_loops
npm install
```

Then run the exercises with:

```bash
node conditionalStatements_loops.js
node ../data_types_operators/datatypes_operators.js
node ../variables/variables.js
```
